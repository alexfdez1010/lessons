An **activation function** takes the neuron's weighted sum `z` and reshapes it
into an output. Without one, a deep network would collapse into a single linear
layer — no matter how many layers you stack. Activations are what let networks
learn curves, not just straight lines.

## Why non-linearity matters

Stacking linear layers is pointless: the composition of linear functions is
still linear. A non-linear activation between layers is what gives the network
its expressive power.

<Callout variant="warning" title="No activation = no depth">
  Remove the non-linear activation and a 50-layer network behaves exactly like a
  single layer. The depth only helps because activations bend the space between
  layers.
</Callout>

## The classic three

<StepThrough
  client:visible
  steps={[
    { title: 'Sigmoid', body: 'Squashes any input into (0, 1). Great for probabilities, but saturates and can stall learning for large inputs.' },
    { title: 'Tanh', body: 'Like sigmoid but centered at zero, mapping to (-1, 1). Usually trains better than sigmoid in hidden layers.' },
    { title: 'ReLU', body: 'Outputs max(0, z): zero for negatives, linear for positives. Cheap, fast, and the default for most modern networks.' },
  ]}
/>

  <Callout variant="tip" title="When in doubt, ReLU">

With weights, a bias, and a non-linear activation, a single neuron is complete.
Wire many together and you have a network capable of learning almost anything.
